# 📊 Modern Dashboard UI - Easysofts Assignment

This is a high-performance, modern Dashboard UI built as part of the **Frontend Developer** assignment for **Easysofts (a sister concern of Easy Fashion Limited)**. The project focuses on clean architecture, the latest React features, and a seamless user experience.

## 🚀 Tech Stack

The project leverages a cutting-edge tech stack for speed, type safety, and maintainability:

- **Framework**: [Next.js 16.1.6](https://nextjs.org/) (App Router)
- **Library**: [React 19.2.3](https://react.dev/) (Latest Release)
- **UI Components**: [Ant Design 6.3.1](https://ant.design/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/) with `tailwind-merge` and `clsx`
- **Icons**: `Lucide React` & `@ant-design/icons`
- **Charts**: `Recharts` for data visualization
- **Runtime**: [Bun](https://bun.sh/) (Fastest JavaScript runtime)
- **Theming**: `next-themes` for Dark/Light mode synchronization

## ✨ Key Features

- **Dark & Light Mode**: Fully integrated theme switching using `next-themes` and Ant Design's `ConfigProvider`.
- **Interactive Visuals**: Data visualization implemented with `Recharts`.
- **Modern Layout**: A clean, sidebar-based navigation structure typical of enterprise dashboards.
- **Performance First**: Optimized using Next.js Server Components and Bun's lightning-fast build process.
- **Clean Code**: Modular component structure and strict TypeScript implementation.

## 🛠️ Installation & Setup

Follow these steps to run the project locally using **Bun**:

1. **Clone the repository:**

```bash
git clone https://github.com/cseswapon/dashboard_ui
cd dashboard_ui

```

2. **Install dependencies:**

```bash
bun install

```

3. **Run the development server:**

```bash
bun dev

```
